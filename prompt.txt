Build a multi-platform MACD Divergence Trading Strategy system — a Next.js web dashboard, Pine Script indicator, MQ5 indicator & Expert Advisor, and Python indicator with backtester.

## Core Algorithm (shared across all platforms)

Detects 4 divergence types between price and MACD using swing-point analysis:

| Type | Price Pattern | MACD Pattern | Signal | Context |
|------|--------------|--------------|--------|---------|
| Regular Bullish | Lower Low | Higher Low | BUY | Reversal in downtrend |
| Regular Bearish | Higher High | Lower High | SELL | Reversal in uptrend |
| Hidden Bullish | Higher Low | Lower Low | BUY | Continuation in uptrend |
| Hidden Bearish | Lower High | Higher High | SELL | Continuation in downtrend |

**Detection steps:**
1. Calculate MACD: Fast EMA (12) - Slow EMA (26), Signal EMA (9)
2. Find swing highs/lows on price and MACD — a bar is a swing high if strictly higher than all neighbors within lookback (default: 5 bars)
3. Track last two swing highs and lows for both price and MACD
4. Compare consecutive swings — if price makes new extreme but MACD doesn't confirm, it's a divergence
5. Draw connecting lines on MACD panel between the two swing points, with labels

## Platform 2: Pine Script Indicator

File: `MACD Divergences_lines.pine`

TradingView indicator (Pine Script v6, overlay=false) that:
- Plots MACD line, Signal line, and Histogram with 4-color scheme
- Detects swing highs/lows on price and MACD using configurable lookback
- Draws divergence lines (solid for Regular, dashed for Hidden) connecting MACD swing points
- Adds text labels at line midpoints
- Color coding: green (Regular Bullish), red (Regular Bearish), orange (Hidden Bullish), purple (Hidden Bearish)
- Alert conditions for all 4 divergence types

**Inputs:** Fast Length (12), Slow Length (26), Signal Smoothing (9), Swing Lookback (5), color settings, alert toggles.

## Platform 4: MQ5 Expert Advisor

File: `MACD_Divergence_EA.mq5`

MetaTrader 5 automated trading EA using the divergence strategy:

**Entry rules:**
- Regular Bullish / Hidden Bullish → Long (BUY)
- Regular Bearish / Hidden Bearish → Short (SELL)
- Signals only processed on new bar close
- Only one position at a time

**Exit rules:**
- Stop Loss: configurable % from entry (default 3%)
- Take Profit: configurable % from entry (default 6%)
- Time exit: max hold bars (default 20)
- Checks exits every tick

**Features:**
- Position sizing from risk % of balance (risk / SL distance)
- Auto SL/TP on order submission
- Magic number filtering for multi-EA compatibility
- Max slippage control
- Uses CTrade class for order management
- iMACD handle for indicator data

**Inputs:** Fast/Slow/Signal lengths, Swing Lookback, Risk %, SL %, TP %, Max Hold Bars, Allow Short, Max Slippage, Magic Number.

## Platform 5: Python Indicator

File: `macd_divergences.py`

Standalone Python module + CLI that:
- Fetches OHLC data via yfinance
- Calculates MACD/Signal/Histogram using EMA
- Detects all 4 divergence types using swing-point logic
- Plots a 2-panel chart: price on top, MACD histogram + lines + divergence annotations below
- Divergence lines: solid (Regular), dashed (Hidden), color-coded
- Annotates divergence labels at line midpoints with colored boxes
- Exports `run()` function for use as a library

**Dependencies:** yfinance, matplotlib, numpy, pandas

```python
from macd_divergences import run
run("TSLA", period="1y", interval="1d", swing_lookback=5)
```

## Platform 6: Python Backtest

File: `backtest.py`

Backtests the MACD divergence strategy on historical data:

**Strategy:**
- Long on Regular Bullish / Hidden Bullish divergences
- Short on Regular Bearish / Hidden Bearish divergences (configurable)
- Exit: SL/TP/Time, whichever hits first

**Config (BacktestConfig dataclass):**
- Initial capital: $100,000
- Position size: 100% of capital
- SL: 3%, TP: 6%
- Max hold: 20 bars
- Commission: 0.1% round-trip
- Allow shorts: true

**Output:**
- Trade list with entry/exit prices, P&L, exit reason
- Performance metrics: total return, win rate, profit factor, Sharpe ratio, max drawdown, expectancy
- Per-divergence-type breakdown
- Exit reason breakdown (SL/TP/TIME)
- Equity curve plot with trade entry/exit markers on price chart
- Saves plot to `backtest_results.png`

**AAPL 2Y Backtest Results:**
- Return: +153.64%
- Sharpe: 2.22
- Max Drawdown: 6.94%
- Win Rate: 76.2% (16/21)
- Profit Factor: 6.45
- Hidden divergences: 100% win rate (10/10)
