Build a Next.js web application called "MACD Divergence Trading Strategy" — a professional technical analysis dashboard that identifies and displays MACD divergence patterns across multiple financial instruments, with an AI-powered chat assistant for trade analysis.

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4 with shadcn/ui components
- Framer Motion for animations
- Lucide React for icons
- Axios for HTTP requests
- Google Fonts: Geist Sans and Geist Mono

## Theme
Dark trading dashboard with a deep navy/black background (#020617). All panels use #0E1223 with subtle white/5 borders. Primary accent colors: emerald (bullish), rose (bearish), cyan (hidden bullish), orange (hidden bearish), blue (AI chat).

## Pages
Single-page application with 3 tabs: Dashboard, Trade Setups, and Divergence Guide. Tab switching via animated transitions. Mobile has a fixed bottom navigation bar.

## Features

### Top Navigation Bar
- Sticky header with logo (SVG waveform icon), app title "MACD Divergence Pro", tab navigation, and pair selector
- Pair selector buttons: AUD/USD, XAU/USD, ETH/USD, BTC/USD
- Mobile hamburger menu for tab navigation
- Bell icon for notifications

### Dashboard Tab
- 4 summary cards showing counts for each divergence type (Regular Bullish, Regular Bearish, Hidden Bullish, Hidden Bearish) with average probability percentages
- A full-width "Ask AI Assistant" button that opens the chat widget
- Price overview panel for the selected pair showing:
  - Pair icon and name with a signal badge (Strong Sell/Sell/Buy)
  - Current price with percentage change
  - Day range, 52-week range, RSI or Volume
  - Animated price ticker that simulates small price movements every 2 seconds
- All Divergence Signals grid showing each active setup as a clickable row with: direction icon, pair name, divergence type badge, probability, and target price. Clicking navigates to Trade Setups tab.

### Trade Setups Tab
- Filter bar with buttons: All Types, Regular Bullish, Regular Bearish, Hidden Bullish, Hidden Bearish
- Each setup is a card showing:
  - Header: direction icon, pair, BUY/SELL badge, divergence type badge, confidence badge (HIGH/MEDIUM/LOW)
  - Current price
  - Analysis paragraph
  - 3 info boxes: Price Action, MACD Behavior, Confirmation criteria
  - Win Probability bar with percentage
  - Win Rate, Expected Return, Risk Amount stats
  - Trade Levels grid: Entry, Stop Loss, TP1, TP2, R:R ratio (calculated as |TP - Entry| / |SL - Entry|)
  - Risk bar: a horizontal bar split between red (SL distance) and green (TP distance)
- Risk Disclaimer banner at the bottom

### Divergence Guide Tab
- "MACD Divergence Masterclass" header
- Key Difference section comparing Regular (reversal) vs Hidden (continuation)
- 4 cards for each divergence type, each containing:
  - Icon and name with reliability rating
  - SVG mini-chart showing the pattern (price line on top, MACD line on bottom, with labeled peaks/troughs)
  - Description and signal
- Visual Pattern Reference: larger SVG diagrams showing Regular Bullish, Regular Bearish, Hidden Bullish, Hidden Bearish with price/MACD lines, labeled LL/HL/HH/LH points, and signal arrows
- Trading Rules grid: 6 rules (Regular=Reversal, Hidden=Continuation, Higher TF=Stronger, Wait for MACD Cross, Volume Confirmation, Risk:Reward 1:2+)

## AI Chat Assistant
### Frontend (ai-chat.tsx)
- Floating blue/cyan gradient button fixed to bottom-right corner, toggles chat open/close
- Animated transitions using Framer Motion (scale, rotate, opacity)
- Chat panel: full-screen on mobile, 800x500px floating panel on desktop with rounded corners
- Header with bot icon, "MACD AI Assistant" title, green "Online" status indicator, and Sparkles icon
- Message list with user messages on right (blue) and AI messages on left (gray) with bot/user icons
- Auto-scroll to bottom on new messages
- Welcome message from AI on first load
- 5 quick-action preset buttons: "Best setup right now?", "What is MACD divergence?", "Explain Regular vs Hidden", "Best BTC trade?", "Risk management tips"
- Text input with placeholder "Ask about MACD, trades, setups..."
- Enter key sends message, Shift+Enter does not
- Send button disabled during typing or when input is empty
- Typing indicator (3 bouncing dots) while waiting for response
- Custom DOM event listener for "open-ai-chat" to open from the dashboard button

### Backend API (POST /api/chat)
- Proxies requests to NVIDIA API endpoint
- System message contains all trade setup data, divergence definitions, and trading rules
- Model: moonshotai/kimi-k2.6
- Parameters: max_tokens 2048, temperature 0.7, top_p 0.9, stream false
- Auth via Bearer token in Authorization header
- 30-second timeout via Axios
- Returns { reply: "..." } on success, { error: "..." } on failure
- Environment variables needed:
  - NVIDIA_BASE_URL (e.g., https://integrate.api.nvidia.com/v1/chat/completions)
  - NVIDIA_API_KEY
  - NVIDIA_MODEL (moonshotai/kimi-k2.6)

## Animations
- All cards and sections use staggered Framer Motion entrance animations (opacity 0->1, y 10->0)
- Tab switching uses AnimatePresence with fade transitions
- Price ticker updates every 2 seconds with simulated small random changes
- Floating chat button has hover scale (1.05) and tap scale (0.95)
- Chat panel has scale/opacity entrance animation
- Typing indicator uses bouncing dots with staggered delays
- Risk bar is a horizontal bar split between red (stop loss distance) and green (take profit distance)

## Mobile Responsive
- Bottom navigation bar with 3 tabs on mobile
- Price selector scrolls horizontally on small screens
- Cards stack vertically on mobile
- Chat widget goes full-screen on mobile
- Safe area padding for notch devices
